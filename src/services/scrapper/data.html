Les missions du poste

Notre agence GESER-BEST Toulouse recherche un.e Développeur Java SpringBoot expérimenté H/F
- Démarrage : ASAP
- Contexte SAFe

Missions
- Développement des web services REST en JAVA avec des pratiques de qualité
- Revues de code, réalisation des tests unitaires et d'intégration
- Optimisation des performances Java, multithreading, gestion de mémoire
- Proposition des actions stratégiques visant à améliorer l'architecture existante et optimiser les performances globales des produits

Compétences requises
- Expérience de 4 ans minimum dans le développement logiciel en Java, avec une spécialisation dans le Framework Spring / Spring Boot
- Expérience pratique avec les outils de gestion de versions tels que Git
- Connaissance de l'environnement Airbus (minimum 3 ans)
- Capacité à travailler efficacement en équipe, avec d'excellentes compétences en communication
- Programmation : Java, SpringBoot, Hibernate, Maven, API REST, SQL, Python
- Compétences en mathématiques appliquées son appréciées
- Connaissances opérationnelles et pratiques sur les technologies AWS, DEVOPS son appréciées
- Anglais : Bon niveau

Nos avantages
- Gestion à taille humaine
- Suivi de mission régulier
- Accompagnement pour une montée en compétence rapide
- Mobilité géographique possible
- Evènements afterwork / teambuilding /caritatifs
- Politique de cooptation
- Offres CSE
- Process d'intégration
- Formation continue
